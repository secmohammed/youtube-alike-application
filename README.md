# youtube-alike-application
This application is made with Laravel & Nuxtjs (RestAPI)

Technologies Used : 
- Unit Testing 
- MailTrap
- Telestream
- AWS
- ADR Design Pattern Instead of MVC
- JWT Auth
- Aloglia Search
- Comments, Replies
- Likes & Dislikes For Videos.
- Subscription


The application is still in progress and under-development
